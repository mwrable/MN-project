Section 1 Overview
Originator Minnesota Legislative Coordinating Commission - Geographic Information Services Office (LCC- GIS)
Title Congressional District Boundaries, Minnesota, 2012
Abstract This is a polygon data set of the 2012 Congressional District Boundaries. It displays district numbers with member names. Includes Congressional districts associated census attribute data. It was derived from U.S. Census Bureau 2010 TIGER/Line files as a result of 2010 redistricting efforts.
Purpose This data was created to display current Congressional District boundaries and to allow geographical review/analysis adhereing to MN Congressional District boundaries.
Time Period of Content Date 2012
Currentness Reference This file is current as of 2010 Census Bureau redistricting efforts. Boundary updates are completed every decennial census.

Section 2 Data Quality
Attribute Accuracy Attributes represent census block demographic information
Logical Consistency None
Completeness Complete Statewide
Horizontal Positional Accuracy No assessment completed.
Vertical Positional Accuracy No assessment completed.
Lineage A block equivalency file for Congressional Districts was received from the Minnesota Supreme Court Special Redistricting Panel. Using Maptitude Software a Congressional Districts boundary file was created in Caliper format. This file was then converted into a shapefile. This database of Congressional District Boundaries was developed from U.S. Census Bureau 2010 TIGER/Line files. The attributes associated with the Congressional District boundaries represent census demographic information. Certain attribute fields were added to calculate percentages based on the census data. This was completed by dissolving census block boundaries to the established Congressional District boundaries.
